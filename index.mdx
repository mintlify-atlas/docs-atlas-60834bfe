## What is Data Analyzer?

Data Analyzer is a lightweight Python desktop application that lets you load CSV files and instantly generate descriptive statistical summaries. Built with `pandas` for data processing and `tkinter` for the graphical interface, it requires no programming knowledge to use.

<Info>
  Data Analyzer is a desktop GUI application — it runs locally on your machine, not in a browser or server.
</Info>

## How it works

<Steps>
  <Step title="Launch the application">
    Run `python src/gui.py` from the project directory to open the GUI window.
  </Step>
  <Step title="Load a CSV file">
    Click **Cargar CSV** and select your file from the file dialog.
  </Step>
  <Step title="Choose analysis type">
    Select **Analizar todo el archivo** to analyze all columns, or **Seleccionar columnas** to pick specific ones.
  </Step>
  <Step title="Review results">
    The statistical summary appears in the scrollable text area, formatted as a readable grid table.
  </Step>
</Steps>
